# 概要
[AutoHotkey](https://www.autohotkey.com/)を用いて、Discordの自分の書き込みを履歴から選択して、順番に削除するマクロです。

# 利用方法
1. Discordを全画面表示に拡大
1. from:<自分の名前>で履歴を表示してください
1. Discordを選択している状態でCtrl+Dを押すとマクロが動作します
1. 中断はESCで行えます
